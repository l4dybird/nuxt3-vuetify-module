# `@l4dybird/nuxt3-vuetify-module`

## Installation

```shell
npm i @l4dybird/nuxt3-vuetify-module
```

## Usage

Add to `modules` (Nuxt 3) in `nuxt.config.ts`:

```js
// Nuxt 3
export default defineNuxtConfig({
  modules: ['@l4dybird/nuxt3-vuetify-module'],
});
```
